iOS Hybrid 小结

1. 实现方案

1.1 JS 调用 Native

  • 通过 WebView.delegate 拦截特定请求。
  • 通过 JavaScriptCore 注册本地方法。
  • 通过 NSURLProtocol 拦截特定 URL。

1.2 Native 调用 JS

  • 通过 WebView 提供的 eval。
  • 通过 JavaScriptCore 获取 JS Function 调用。

2. WebViewJavascriptBridge 源码解读

  • WKWebViewJavascriptBridge，对应新的 WKWebView。
  • WebViewJavascriptBridgeBase，上面两个类的公共实现部分。
