UIView & CALayer

  • CALayer.contents 之所以类型为 id，是因为在 Mac OS 系统上，这个属性可以是 CGImage，也可以是 NSImage。

  • UIView.contentMode 等价于 CALayer.contentGravity。

  • UIView.contentScaleFactor 等价于 CALayer.contentsScale，比如在 Retina 显示屏上，该值设为 2.0，则每 2*2 的像素点对于到 1*1 的逻辑点。

  • UIView.clipsToBounds 等价于 CALayer.masksToBounds

  • CALayer.contentsRect 默认为 {0, 0, 1, 1}，如 {0.25, 0.25, 0.5, 0.5} 表示 contents 的中间部分可见。

  • CALayer.contentsCenter 默认为 {0, 0, 1, 1}，如图。
