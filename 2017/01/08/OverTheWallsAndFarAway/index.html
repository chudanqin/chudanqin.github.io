Over The Walls And Far Away

发表于 2017-01-08

之前买的翻墙服务遭到封杀，需要经常更新服务器地址，很是麻烦。可翻墙毕竟是刚需，只好自己动手了。我选择了亚马逊的AWS（可以免费使用一年，只需要一张信用卡即可）。

首先，需要一个AWS的账号。如果注册中遇到无法接收确认电话的问题，可以提工单给客服（技术顾问是说英文的老外，基本的听说能力即可应付）。

注册成功后需要等待一会才能使用AWS的服务，登录后选择EC2创建一个新实例，然后按照指示完成整个流程即可（注意保存下证书，cloudping.info可以查看下哪个节点延迟比较低）。


通过ssh连接到服务器。

安装shadowsocks（用的node）。

修改下配置文件（/usr/local/lib/node_modules/shadowsocks/config.json）

启动shadowsocks server

由于AWS默认只开启了ssh的端口，所以需要在安全组下开放端口。
